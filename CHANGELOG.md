# Changelog

All notable changes to this project follow [Keep a Changelog](https://keepachangelog.com/en/1.1.0/) and [Semver](https://semver.org/).

## [0.1.0] — 2026-05-01

### Added
- Initial public release.
- `SKILL.md` with trigger description, decision flow (classify → context → effects → output), and anti-patterns.
- `references/effects-catalog.md` — 12 canonical CLT effects with mechanism, conditions, before/after examples.
- `references/decision-trees.md` — expertise × intrinsic load → effects matrix.
- `references/review-checklist.md` — 19-point structured review pass across 5 sections.
- `references/learner-side-strategies.md` — gestures, collaboration, environment; explicitly marked unverified for AI-mediated learning.
- `references/caveats.md` — de Jong (2010) critique on falsifiability of germane load + AI-extrapolation gap.
- `references/sources.md` — primary papers behind every claim.
- `examples/` with three sample outputs: GENERATE (novice notes), REVIEW (existing chapter), DIAGNOSE (overwhelm).
- `README.md`, `ROADMAP.md`, `CONTRIBUTING.md`, MIT `LICENSE`.

### Known limits
- No empirical eval harness yet; effectiveness claims rest on extrapolation from human-learner literature.
- Domain-neutral only; PT/STEM/procedural specifics deferred to sibling skills.
